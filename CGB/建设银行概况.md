# 建设银行2025回顾：45.63万亿资产与"因建而生"的71年进化论

> 从1954年管理国家基建拨款到45.63万亿现代商业银行集团，从"拨改贷"到股改上市20年累计分红超1.4万亿，建设银行的故事就是一部中国国家建设的金融镜像。

---

## 01 一句话读懂建行：71年，"建设"二字的变与不变

**"因建而生、因建而兴"**——这不是一句宣传语，而是建设银行71年发展史的精确概括。

建设银行的历史可以追溯到**1954年10月1日**，成立时的名称是**中国人民建设银行**，当时的职能是"管理和分配根据国家经济计划拨给建设项目和基本建设相关项目的政府资金"，本质上是**财政部下属的国有独资银行兼拨款机构**。

此后历经三次关键跃迁：

- **1979年 → 国务院直属金融机构**：从财政出纳走向"财政职能+银行职能并举"，开办储蓄、工商贷款、住房金融等商业银行业务，奠定向现代商业银行转轨的基础
- **1994年 → 政策性职能剥离**：国家开发银行承接政策性基本建设贷款职能后，建行正式踏上"一心一意办银行"的综合化商业银行之路，1996年更名为**中国建设银行**
- **2004年 → 股份制改造**：建行由国有独资改制为**国家控股的股份制商业银行**（汇金公司为控股股东），2004年9月17日**中国建设银行股份有限公司**正式成立；**2005年10月27日H股在港交所挂牌（00939），2007年9月25日A股在上交所挂牌（601939）**——在四大行中**率先完成海外上市**

2025年恰是**建行成立71周年、股改上市20周年**。站在这一节点回望，"建设"二字的时代内涵已从"基本建设拨款"扩展为**建设现代化产业体系、建设强大国内市场、建设人民美好生活**。

## 02 财务全景：45.63万亿体量下的"韧性答卷"

2025年，建设银行交出了一份**规模再上台阶、盈利稳中有进、质量持续改善**的成绩单：

| 核心指标 | 2025年末/全年 | 同比变化 |
|---|---:|---:|
| **资产总额** | **45.63万亿元** | **+12.47%** |
| 负债总额 | 41.95万亿元 | +12.68% |
| 发放贷款和垫款净额 | 26.93万亿元 | +7.53% |
| **经营收入** | **7,408.71亿元** | **+1.69%** |
| **净利润** | **3,397.90亿元** | **+1.04%** |
| 归母净利润 | 3,389.06亿元 | +0.99% |
| **不良贷款率** | **1.31%** | **↓0.03pp** |
| 拨备覆盖率 | 233.15% | 基本持平 |
| 核心一级资本充足率 | 14.63% | +0.15pp |
| **资本充足率** | **19.69%** | 持平 |
| 净息差 | 1.34% | -0.17pp |
| ROA / ROE | 0.79% / 10.04% | — |
| 成本收入比 | 29.44% | ↓0.14pp |

几个值得细读的信号：

- **总资产突破45万亿、贷款净额近27万亿**，规模稳居四大行之列，扩表速度（+12.47%）在大型银行中处于偏快区间，反映信贷投放力度
- **净利润增速逐季向好**，经营收入与净利润实现"双增"，净利息收入降幅逐季收窄，**非利息收入占比提升3.65个百分点**，手续费及佣金净收入增幅5.13%——收入结构在优化
- **不良贷款率1.31%连续下降**，拨备覆盖率233%保持充足，证明资产质量并未因扩表而松动
- **核心一级资本充足率14.63%、资本充足率19.69%**，顺利完成向财政部发行A股1,050亿元补充核心一级资本，资本"底盘"极为厚实
- **现金分红比例保持30%**，全年每10股派息3.887元、总额约1,016.84亿元，股改上市20年来**累计分红将超1.4万亿元**

## 03 客户底盘：7.85亿人与1273万户的"国民银行"

建行的特别之处不在于它最大，而在于它的**客户结构与国家经济结构的高度重叠**：

- **个人全量客户7.85亿人**——几乎覆盖了中国一半的人口
- **公司类客户1,273万户**，从央企、地方国企到民营中小微，梯度完整
- **管理个人客户金融资产突破23万亿元**，个人贷款规模达**9万亿元**（其中个人住房贷款5.99万亿元）
- **营业机构14,614个**、员工**378,344位**，虽在推进轻型化，但物理网络仍为综合服务能力提供支撑

这组数字的深层含义是：**建行本质上是一家"国民银行"而非"精英银行"**——它的利润来源既靠对公大客户，也靠海量个人客户的存款基础和按揭基本盘。理解了这一点，就理解了为什么建行在"住房金融""普惠金融""金融科技渠道"三条线上始终走在行业前面。

## 04 业务结构：公司、个人、资金资管与综合化经营的"四梁八柱"

建行的业务架构可以概括为"一本部+多牌照"：

### 公司金融
- 境内公司类贷款**15.69万亿元**（+8.70%），制造业贷款增幅15.83%，战略性新兴产业贷款增幅23.46%
- 积极服务京津冀、长三角、大湾区、成渝等重点区域，**国际业务信贷余额1.45万亿元**，跨境人民币结算量**6.50万亿元**

### 个人金融
- 个人住房贷款**5.99万亿元**，信用卡贷款余额**1.01万亿元**，个人消费贷款增幅**29.41%**
- 住房金融是建行的"基因优势"——它是中国住房制度改革与按揭金融最重要的推动者和参与者

### 资金与资管
- 资金交易、金融市场、资管业务构成利润的重要补充，非利息收入占比持续提升至**22.69%**

### 综合化经营
- 在基金、租赁、信托、保险、期货、养老金、投行等领域拥有**23家主要附属公司**，机构535个，形成"银行+非银"的综合化服务矩阵

## 05 "五篇大文章"：科技5.25万亿、绿色6万亿的战略落子

建行将2025年业务亮点高度浓缩为金融"**五篇大文章**"的执行力：

| 领域 | 关键数据 |
|---|---|
| **科技金融** | 科技贷款余额**5.25万亿元**，承销科创债券719.84亿元，累计设立**28只AIC股权投资试点基金** |
| **绿色金融** | 绿色贷款余额**约6万亿元**，全年境内外发行绿色金融债**超720亿元**（单期最大纪录之一） |
| **普惠金融** | 普惠型小微贷款保持高增，民营经济贷款增幅12.17% |
| **养老金融** | 个人养老金+企业年金服务体系持续扩容 |
| **数字金融** | 手机银行+AI+智慧银行建设持续深化 |

值得注意的是，建行的打法不是"撒胡椒面"，而是**把住房金融的按揭风控能力平移到科创、绿色、普惠领域**——用大数据替代抵押品、用场景嵌入替代物理网点依赖，形成了差异化的风控-业务闭环。

## 06 治理与领导力：张金良×张毅的"董事长-行长"组合

- **董事长：张金良**（1969年生），曾任邮储银行董事长、中国银行副行长，2024年入主建行，财务出身、资本管理见长
- **行长：张毅**，曾任工银澳门董事长、工行副行长、中行副行长，2024年接任建行行长，银行实务与国际化经验丰富

这一组合的特点是**财务纪律+战略执行**的强力咬合：董事长把控资本约束与战略方向（如向财政部增发A股补充核心资本），行长负责经营落地与业绩兑现（如净利润增速逐季修复、成本收入比持续优化）。

治理结构层面，建行遵循**股东大会→董事会（含专门委员会）→高级管理层→内部审计**的现代公司治理链条，董事会下设审计、风险、战略等专业委员会，并在**MSCI ESG评级保持最高等级AAA**。

## 07 历史坐标：从"基本建设出纳"到"全球银行第7"的进化逻辑

| 阶段 | 关键词 | 核心事件 |
|---|---|---|
| 1954–1978 | 财政拨款工具 | 中国人民建设银行成立，管理基建拨款 |
| 1979–1993 | 专业银行雏形 | "拨改贷"，开办储蓄、工商贷款、住房金融 |
| 1994–2003 | 商业化转轨 | 政策性职能剥离→建行更名→"一心一意办银行" |
| 2004–2007 | 股份制+上市 | 股改→H股上市（四大行首家）→A股上市 |
| 2008–至今 | 综合化+数字化 | 非银牌照扩张、"五篇大文章"、AI+渠道 |

这条进化链的底层逻辑其实很简单：**建行每一次转型，都不是自主选择，而是主动对齐国家经济建设的阶段性任务**——从"建厂房"到"建住房"到"建产业"到"建新质生产力"。

## 写在最后

**45.63万亿资产、7.85亿客户、1.31%不良率、30%分红比例、AAA的ESG评级**——这组数字拼在一起，描述的不是一台冷冰冰的金融机器，而是一个与国家经济同频共振了71年的"建设者"。

如果说工商银行的标签是"宇宙行"、农业银行的标签是"三农"、中国银行的标签是"国际"，那么**建设银行最鲜明的身份锚，始终只有一个字：建**。

房子要建、产业要建、城市要建、科技也要建。谁能把"建设"二字从砖头钢筋翻译成金融语言，谁就掌握了这家银行真正的密码。
