# 分众传媒公司概况：电梯"必经场景"的注意力垄断者

> 在中国广告业，真正稀缺的不是创意，而是**无法被跳过的注意力入口**。分众传媒把这件事做到了极致——它不是做内容的平台，而是把全国写字楼的等电梯时间，做成了中国最大规模的城市生活圈媒体网络。

---

## 01 一句话读懂：分众到底"卖"什么？

根据公司官方投资者关系口径，分众传媒当前的主营业务是**生活圈媒体的开发和运营**，核心产品为[楼宇媒体]（电梯电视媒体 + 电梯海报媒体/框架）以及**影院银幕广告媒体** ，覆盖城市主流消费人群的工作、生活、娱乐、消费场景，并整合为"生活圈媒体网络"。

用更直白的话讲：**向上跟物业/业主委员会租下电梯广告位，装上屏幕或海报框架，再向下把"广告时段/广告位"打包卖给品牌方**，赚的是"点位租金 + 硬件运维"与"广告刊例收入"之间的差价。

它不是内容平台，不靠DAU变现，而靠一个物理事实赚钱——**城市主流人群每天必经的封闭空间（电梯等候区/轿厢），用户无法像刷手机那样一键关闭**。

---

## 02 公司基本面快照（2025年报 / 2026Q1）

### 核心财务数据（可核对）

- **2025年全年**：营业收入 **127.59亿元**（同比 +4.05%）；归母净利润 **29.46亿元**；扣非归母净利润 **27.19亿元**；毛利率 **70.48%**；资产负债率 **27.71%**。
- **2026年一季度**：营收 **29.15亿元**（+2.01%），归母净利润 **17.90亿元**（+57.65%）；扣非归母净利润 **10.66亿元**（+7.88%）。

### 一个关键"读财报提醒"：2025年利润为何暴跌42.85%？

2025年归母净利同比大降，市场第一直觉往往是"主业不行了"。但原因非常明确：**对联营公司数禾科技计提长期股权投资减值（合计影响约25.28亿元净利润）**——这是一次性的、非现金的会计调整，不等于主业崩了。

佐证就在2026Q1：剔除处置数禾带来的投资收益扰动后，扣非净利同比约+7.88%，主业仍在走"小幅稳健增长"轨道。公司亦在推进**退出数禾科技投资**，把它从"扰动源"清理出利润表。

## 03 业务结构：楼宇媒体占95%，影院媒体是配角

分众的收入极其聚焦——这既是优势（简单、可预测），也是风险（单一场景依赖）。

**2025年收入构成（年报口径）**：
- **楼宇媒体及其他：120.34亿元，占比约94.32%**
  - 其中"楼宇媒体"口径约120.34亿，"其他媒体及其他"约0.85亿
- **影院媒体：6.39亿元，占比约5.01%**（同比仍在下滑，-7.24%）

**点位口径（更能看出战略方向）**：
- 截至2025年末/2026Q1：自营**电梯电视媒体约122.3万台**（同比仍有增长，数字化屏是重点）
- 自营**电梯海报（框架）约106万～112.9万个**，近年呈主动收缩/优化趋势——本质是**用更少、更好的点位 + 更多数字屏**，替代低效静态框架，拉升效率。

**客户结构是最重要的变化变量**（2025年楼宇媒体分行业）：
- **日用消费品：60.36亿元（占比约47.3%）**，同比 **-16.22%**——消费保守期，部分快消/大促预算收缩
- **互联网：32.08亿元（占比约25.2%）**，同比 **+188.59%**——AI大模型/平台投放把梯媒当作品牌引爆通道

这意味着分众的广告主结构正从"消费品单核驱动"，走向"消费品 + 互联网/AI"双轮驱动，周期性来源也变得更复杂（既有消费周期，也有互联网投放周期）。

## 04 商业模式本质：极简、轻资产、高经营杠杆

### （1）"线下注意力批发商"的盈利公式

```
利润 ≈ 广告刊例价 × 折扣后售卖率 × 点位覆盖 −（物业租金 + 销售/运维/折旧）
```

- **收入端**：对品牌广告主，分众卖的是**城市主流人群高频必经的强制曝光**；刊例价是"标价"，真实成交价取决于谈判与返点，但核心壁垒在于——你想打一线城市中产，分众是绕不开的网络。
- **成本端（最关键）**：最大一块不是设备折旧（屏幕折旧摊销相对可控），而是**媒体资源成本 = 给物业/业委会的点位租金**，这是分众商业模式里最敏感、也最有护城河的一环。

2025年一个值得重视的信号：**媒体资源成本同比下降约14.45%**，叠加结构优化（数字屏占比提升），推动楼宇媒体毛利率升至 **69.96%**，综合毛利率达 **70.48%**。

### （2）轻资产 + 极低CAPEX = 现金牛体质

分众不建楼、不买房，核心是签长期点位合约、铺硬件、做运维与销售网络。CAPEX常年很小，但经营现金流很强：2025年**经营活动现金流净额约72.09亿元**，自由现金流充裕。这让它可以维持极高分红率（2025年分红总额远超表观归母净利，靠的是现金流而非"印钞"）。

### （3）高经营杠杆：顺风时利润弹、逆风时更痛

因为大量成本（租金合约、销售网络）相对固定，所以营收小幅波动会被放大到利润端——这也是为何分众利润表在宏观/广告预算下行期看起来"更吓人"，反之复苏期修复也快。

## 05 护城河：为什么"新玩家很难打死它"

分众的壁垒经常被简化成"点位多"，但更准确地说，是**四条互相咬合的护城河**：

1. **点位规模 + 排他合约 → 物理性壁垒**  
   一二线核心写字楼/中高档社区的优质电梯资源是有限的、不可再生的；长期独家协议让竞品"有钱也难进场"。

2. **双边网络效应 → 鸡生蛋蛋生鸡**  
   点位越多→覆盖人群越准→越能说服大客户投；客户越多→现金流越强→越能续租/升级点位→进一步拉开与对手差距。

3. **大客户服务能力（含江南春本人的品牌洞察体系）**  
   大客户不只买屏幕，还买"策略+创意共创+效果归因"。这与分众二十多年沉淀的销售组织能力绑定在一起，竞争对手短期内很难复制（能买到屏幕，但买不到这套服务体系）。

4. **财务安全垫：低杠杆 + 强现金流**  
   资产负债率不到30%，现金流充沛，意味着价格战即便发生，分众也比小玩家更"耗得起"。

**但护城河不是无懈可击**：最现实的软化力量来自两点——（a）物业租金长期谈判压力；（b）监管/反垄断对"排他协议"边界的重估（2026年已有围绕梯媒排他行为的诉讼/争议进入公众视野，是否影响合约范式仍有待观察）。

## 06 战略动向：收购新潮、出海、"数字屏替代框架"

- **拟收购新潮传媒（约83亿元估值）**：若最终通过审查完成整合，梯媒格局将从"一超一强"走向更集中的"绝对寡头"，定价权更强，但也必然带来更严格的**反垄断审视**。
- **点位迭代**：继续把静态海报框架替换为可程序化投放的数字屏，提升售卖率与千人成本效率。
- **海外布局**：已进入韩国、东南亚、日本、阿联酋等市场，韩国/新加坡据报道已实现盈利，但整体海外仍处在"先铺网络、再证明规模利润"的阶段。
- **AI + 品效协同**：公司与支付宝合作推进"电梯碰一下"等线下线上联动，并推动基于AI的营销智能体"众小智"落地，方向是让梯媒从"品牌曝光"走向"可追踪、可转化"。

## 07 投资视角：分众该怎么被定价——它不是成长股，而是"垄断现金流 + 高分红 + 周期波动"

把分众放进正确框架里，就不容易被表面数字误导：

- **它的核心价值** = 对城市主流人群"必经场景"的渠道垄断 + 极轻资产结构 + 充裕自由现金流 → 高分红能力。
- **它的核心风险** = 广告主预算（消费周期 + 互联网投放周期）波动 + 物业租金刚性上升 + 排他合约的法律/商誉风险 + 影院媒体边缘化（但影院本来只占5%）。
- **估值关键**：别用"表观PE（含一次性减值）"拍脑袋，而应以**剔除非经常性扰动后的常态化利润/现金流**为锚，再看股息率与安全边际。

---

### 结语

分众传媒的公司概况可以用一句话收束：**中国最大、也可能是唯一的"全国性电梯场景媒体网络"**——生意极简（租点位→卖曝光），壁垒极硬（点位+合约+大客户网络），财务极轻（低CAPEX、高现金流、低杠杆），代价是它对宏观广告预算的周期性高度敏感，且护城河的"排他性"必须在法律与商业之间长期走钢丝。

对投资者而言，分众更像一台**带周期波动的线下注意力印钞机**：你赚的不是技术爆发的贝塔，而是它在梯媒这个狭窄入口里持续把持的定价权与分红回报——前提是你能忍受它利润表随广告周期的起落，并用现金流/常态化盈利而非单次净利润去定价。
